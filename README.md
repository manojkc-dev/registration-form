# Student Registration Form

A clean, modern, and fully responsive **Student Registration Form** built using HTML5 and styled with **Bootstrap 5**. This project demonstrates a mobile-first, multi-column grid layout that adapts seamlessly across all device screen sizes.

## 🚀 Features

*   **Fully Responsive Grid:** Uses Bootstrap's layout system (`col-12 col-md-6`) to automatically stack fields on mobile screens and split into two columns on desktop viewports.
*   **Comprehensive Data Input:** Collects essential student details including Name, Email, Contact Number, Address, Country, Date of Birth, and Gender.
*   **File Upload & Dropdown:** Integrated fields for uploading a profile picture and selecting preferred programming subjects via a styled dropdown menu.
*   **Form Control Styling:** Features a clean, centered layout with a bold blue title header and a full-width submission button.
*   **Zero Setup Overhead:** Utilizes official Bootstrap 5 CDNs for styling and scripts, making it incredibly lightweight and fast to load.

## 🛠️ Built With

*   **HTML5** - Markup structure and native form inputs.
*   **Bootstrap v5.3.8** - Responsive grid system, typography, and component styling.

## 📦 Form Fields Overview

| Section | Input Type | Description |
| :--- | :--- | :--- |
| **Personal Details** | Text, Date, Radio | First Name, Last Name, Date of Birth, Gender Selection |
| **Contact Info** | Email, Text | Email Address, Contact Number, Full Address, Country |
| **Academics / Profile**| File, Select | Profile Picture upload, Subject selection (Python, Java, JS, etc.) |
| **Security** | Password | Password and Confirm Password matching fields |
